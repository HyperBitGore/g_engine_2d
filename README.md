# g_engine_2d
 Win32 header only game engine using modern OpenGL

 Current Features
  - Modern OpenGL Context creation
  - TTF font Loading and Rendering
  - 2D Image drawing
  - 2D Primitive drawing
  - PNG and BMP loading
  - Audio Support
     - Audio playing
     - WAV file loading
     - Stream audio from files
     - Play audio from seperate thread
